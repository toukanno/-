# trends-repo

トレンド情報の収集・整理・発信を行うためのリポジトリ雛形です。

技術・デザイン・ビジネス・カルチャーなど各ジャンルのトレンドを時系列（日次・週次・月次）とカテゴリ別の両軸で管理します。

## ディレクトリ構成

```
trends-repo/
├── README.md          # 本ファイル
├── index.html         # トレンドダッシュボード
├── daily/             # 日次トレンド記録
├── weekly/            # 週次トレンドまとめ
├── monthly/           # 月次トレンドまとめ
├── categories/        # カテゴリ別トレンド
│   ├── tech.md        # 技術トレンド
│   ├── design.md      # デザイントレンド
│   ├── business.md    # ビジネストレンド
│   └── culture.md     # カルチャートレンド
└── assets/            # 画像・データ・スクリーンショット
```

## 運用フロー

1. **日次収集**: `daily/YYYY-MM-DD.md` に当日キャッチしたトピックを箇条書き
2. **週次まとめ**: 毎週末、`weekly/YYYY-Www.md` に7日分の要点を整理
3. **月次レポート**: 月初に `monthly/YYYY-MM.md` へ月間サマリを作成
4. **カテゴリ更新**: 重要トピックは `categories/` 配下の該当ファイルに転記
5. **ダッシュボード確認**: `index.html` をブラウザで開いて全体像を俯瞰

## ファイル命名規則

| 種別 | パス例 |
|------|--------|
| 日次 | `daily/2026-04-21.md` |
| 週次 | `weekly/2026-W17.md` |
| 月次 | `monthly/2026-04.md` |
| カテゴリ | `categories/tech.md` |

## 記事テンプレート（日次）

```markdown
# YYYY-MM-DD トレンドメモ

## Tech
- トピック名 — ひとことメモ (出典URL)

## Design
- トピック名 — ひとことメモ (出典URL)

## Business
- トピック名 — ひとことメモ (出典URL)

## Culture
- トピック名 — ひとことメモ (出典URL)
```

## ダッシュボード

`index.html` をブラウザで開くと、カテゴリ別の最新トレンドと更新履歴を俯瞰できます。外部依存なしの単一HTMLで動作します。
