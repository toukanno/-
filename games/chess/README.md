# チェス (Chess)

ブラウザで遊べる二人対戦チェスゲームです。

## 遊び方

1. `index.html` をブラウザで開く
2. 駒をクリックして選択し、移動先をクリック
3. 有効な移動先は緑色でハイライト表示されます

## 機能

- 全チェスルール対応（キャスリング、アンパッサン、プロモーション）
- チェック・チェックメイト検出
- 棋譜（移動履歴）パネル
- 取った駒の表示
- ターン表示
- 新しいゲーム / 一手戻る ボタン
- モバイル対応レスポンシブデザイン

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `game.js` - ゲームロジック
