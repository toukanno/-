# ブロック崩し (Breakout)

A retro-style block breaker game built with HTML5 Canvas.

## How to Play

Open `index.html` in a browser.

### Controls
- **Mouse**: Move to control paddle
- **Touch**: Drag to control paddle (mobile)
- **Arrow Keys / A-D**: Move paddle left/right
- **Space / Click**: Launch ball, start game

### Features
- Multiple rows of colored bricks with different point values
- 3 lives per game
- Level progression with increasing difficulty (faster ball, more bricks)
- Power-ups:
  - **W (wider paddle)**: Expands the paddle temporarily
  - **M (multi-ball)**: Splits into 3 balls
  - **S (slow ball)**: Slows the ball temporarily
- Japanese retro arcade UI
- Mobile touch support
