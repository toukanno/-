# オセロ / Othello (Reversi)

ブラウザで遊べるオセロゲームです。

## 遊び方

`index.html` をブラウザで開いてください。

## 機能

- 8x8 の盤面（クラシックな緑のデザイン）
- 二人対戦モード（ローカル）
- AI 対戦モード（貪欲法アルゴリズム）
- 有効な手をドットで表示
- 石を裏返すアニメーション
- スコア表示（黒 vs 白）
- 手番表示
- パス検出（有効な手がない場合）
- ゲーム終了時の勝敗表示
- ニューゲームボタン
- モバイル対応レスポンシブデザイン
- 日本語 UI

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `game.js` - ゲームロジック
- `README.md` - このファイル
