# 数独 (Sudoku)

A complete browser-based Sudoku game with a clean modern UI, fully in Japanese.

## Features

- 9x9 grid with clearly marked 3x3 boxes
- Three difficulty levels: 簡単 (Easy), 普通 (Medium), 難しい (Hard)
- Click a cell and press a number key (1-9) to fill it in; press Delete/Backspace to clear
- Highlights all matching numbers on the board when a cell is selected
- Errors shown in red (conflicts with row, column, or box)
- Timer tracks how long you have been playing
- ヒント (Hint) button fills in one correct cell
- 確認 (Check) button validates your solution
- 新しいゲーム (New Game) button starts a fresh puzzle
- Mobile responsive layout with touch support

## How to Play

Open `index.html` in any modern browser.
