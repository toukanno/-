# ハングマン (Hangman)

ブラウザで遊べるハングマン（単語推測ゲーム）です。

## 遊び方

1. `index.html` をブラウザで開く
2. カテゴリのヒントを参考に、アルファベットを選んで単語を当てる
3. 6回間違えるとゲームオーバー

## 機能

- SVGによるハングマン描画（頭・胴体・両腕・両脚 = 6回）
- カテゴリ付きの英単語リスト（動物、果物、国名など）
- A-Z のレターボタン（正解は緑、不正解は赤）
- キーボード入力対応
- 勝敗スコア記録
- モバイルレスポンシブデザイン
- 日本語UI

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `game.js` - ゲームロジック
