# サイモンゲーム (Simon Says)

A sleek, modern Simon Says memory game with Japanese UI.

## How to Play

1. Press the スタート (Start) button to begin.
2. Watch the sequence of colors the computer plays.
3. Repeat the sequence by clicking (or tapping) the colored buttons.
4. Each round adds one more color to the sequence.
5. The game speeds up as you progress.
6. If you make a mistake, the game is over.

## Features

- 4 colored buttons (red, green, blue, yellow) in a circular arrangement
- Sound effects using the Web Audio API (no external files needed)
- Score display showing current round
- High score saved to localStorage
- Speed increases as rounds progress
- Game over screen with final score
- Fully responsive design for mobile and desktop
- Japanese UI text

## Running

Open `index.html` in any modern web browser. No build step or server required.
