# マインスイーパー (Minesweeper)

クラシックなマインスイーパーゲームのWebブラウザ版です。

## 遊び方

- **左クリック**: マスを開く
- **右クリック**: 旗を立てる／外す
- **スマイルボタン**: ゲームをリスタート

## 難易度

| レベル | サイズ | 地雷数 |
|--------|--------|--------|
| 初級   | 9x9    | 10     |
| 中級   | 16x16  | 40     |
| 上級   | 30x16  | 99     |

## ルール

1. 数字はその周囲8マスにある地雷の数を示します
2. 地雷のないマスを全て開けばクリアです
3. 最初のクリックは必ず安全です
4. 空白マスを開くと、周囲の空白マスも自動的に開きます

## 起動方法

`index.html` をブラウザで開いてください。

```
open index.html
```
