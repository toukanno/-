# Desktop Apps Collection

デスクトップアプリケーション20個のコレクション。

## アプリ一覧

| # | アプリ名 | 説明 | 技術 |
|---|---------|------|------|
| 01 | Calculator | 関数電卓 | Electron |
| 02 | Todo App | タスク管理アプリ | Electron |
| 03 | Text Editor | テキストエディタ | Electron |
| 04 | Image Viewer | 画像ビューア | Electron |
| 05 | Music Player | 音楽プレイヤー | Electron |
| 06 | Weather App | 天気予報アプリ | Electron |
| 07 | File Manager | ファイルマネージャー | Electron |
| 08 | Paint App | ペイントアプリ | Electron |
| 09 | Chat App | チャットアプリ | Electron |
| 10 | Calendar | カレンダーアプリ | Electron |
| 11 | Password Manager | パスワード管理 | Electron |
| 12 | Markdown Editor | Markdownエディタ | Electron |
| 13 | Pomodoro Timer | ポモドーロタイマー | Electron |
| 14 | Screen Recorder | スクリーンレコーダー | Electron |
| 15 | Clipboard Manager | クリップボード管理 | Electron |
| 16 | Color Picker | カラーピッカー | Electron |
| 17 | System Monitor | システムモニター | Electron |
| 18 | Note Taking | メモ帳アプリ | Electron |
| 19 | PDF Viewer | PDFビューア | Electron |
| 20 | Video Player | 動画プレイヤー | Electron |

## セットアップ

各アプリのディレクトリに移動して以下を実行:

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```
