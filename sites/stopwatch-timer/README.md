# ストップウォッチ & タイマー

A full-featured stopwatch and timer web application with a beautiful dark UI, Japanese interface, and mobile-responsive design.

## Features

### ストップウォッチ (Stopwatch)
- Large digital display showing HH:MM:SS.ms
- Start / Stop / Reset / Lap controls
- Lap times list with lap number and split time

### タイマー (Timer)
- Hour / minute / second input wheels
- Start / Pause / Reset controls
- Audio alarm when timer reaches zero (Web Audio API)
- Preset timers: 1min, 3min, 5min, 10min, 30min

### General
- Two-tab interface for switching between stopwatch and timer
- Full-screen mode
- Beautiful large typography on a dark background
- Mobile responsive
- Japanese UI throughout

## Usage

Open `index.html` in any modern browser. No build step or server required.

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript (ES6+)
- Web Audio API for timer alarm
