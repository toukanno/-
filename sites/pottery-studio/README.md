# 土と炎 陶芸工房 - Tsuchi to Honoo Pottery Studio

A website for a Japanese pottery studio offering hands-on ceramic workshops and courses.

## Overview

"土と炎 陶芸工房" (Earth & Flame Pottery Studio) is an artisan pottery workshop located in Kyoto, Japan. The studio offers various ceramic experiences for beginners through advanced potters, including hand-building, wheel throwing, painting, and regular courses.

## Files

- `index.html` - Main HTML page with all sections
- `style.css` - Stylesheet with earthy, warm color palette and responsive design

## Sections

1. **Header & Navigation** - Fixed navigation with smooth scroll links
2. **Hero** - Earthy gradient banner (#8D6E63 to #D7CCC8) with studio tagline
3. **Workshop Types** - Four experience types:
   - 手びねり体験 (Hand-building)
   - ろくろ体験 (Wheel throwing)
   - 絵付け体験 (Painting/glazing)
   - 定期コース (Regular courses)
4. **Works Gallery** - Showcase of studio pottery pieces
5. **Artist/Instructor Profile** - Studio master and instructors
6. **Price List** - Pricing for all workshops and courses
7. **Access & Schedule** - Location map placeholder, business hours, directions
8. **Contact/Reservation Form** - Booking form with fields for name, email, phone, date, workshop type, and message
9. **Footer** - Studio info, quick links, social media, copyright

## Design

- **Color Palette**: Earthy browns, warm terracotta, soft clay tones (#8D6E63, #D7CCC8, #5D4037, #BCAAA4, #4E342E, #A1887F)
- **Typography**: "Noto Serif JP" for headings, "Noto Sans JP" for body text
- **Layout**: Mobile-first responsive design
- **Style**: Warm, artisan aesthetic inspired by Japanese wabi-sabi principles

## Usage

Open `index.html` in a web browser. No build step or server required.
