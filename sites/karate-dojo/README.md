# 武道館 空手道場 - Budokan Karate Dojo Website

A bold, responsive website for a traditional Japanese karate dojo.

## Structure

- `index.html` - Main HTML page with all sections
- `style.css` - Stylesheet with responsive design and martial arts aesthetic

## Sections

1. **Header** - Navigation bar with dojo name and links
2. **Hero** - Full-screen hero with black-to-red gradient
3. **Classes** - Four class tiers: Kids, Junior, General, Senior
4. **Philosophy** - Dojo philosophy rooted in tradition
5. **Schedule** - Weekly class timetable
6. **Instructors** - Instructor profile cards
7. **Pricing** - Membership pricing plans
8. **Gallery** - Photo gallery placeholders
9. **Contact** - Contact form
10. **Footer** - Footer with links and info

## Usage

Open `index.html` in any modern browser. No build step or dependencies required.
