# オートガレージ信頼 - Used Car Dealer Website

A professional Japanese used car dealership website for **オートガレージ信頼** (Auto Garage Shinrai).

## Overview

A single-page, mobile-responsive website built with plain HTML and CSS. The design emphasizes trustworthiness and professionalism appropriate for an automotive business.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main HTML document containing all page sections |
| `style.css` | All styles including layout, responsive breakpoints, and animations |

## Sections

1. **Header** - Sticky navigation bar with logo and links
2. **Hero** - Bold gradient banner (#263238 to #455A64) with call-to-action
3. **Search Filter** - Filter cars by maker, price range, and model year
4. **Featured Cars** - 6-card grid showcasing inventory with price, year, and mileage
5. **Services** - Five service offerings (中古車販売, 車検・整備, 買取査定, カスタム, 保険)
6. **Why Choose Us** - Trust signals and differentiators
7. **Customer Reviews** - Testimonial cards
8. **Access Info** - Dealership location, hours, and embedded map placeholder
9. **Contact Form** - Inquiry form for customers
10. **Footer** - Links, copyright, and company info

## Usage

Open `index.html` in any modern browser. No build tools or server required.

## Browser Support

Tested for modern evergreen browsers (Chrome, Firefox, Safari, Edge). Responsive from 320px to large desktop viewports.
