# 和漢堂 鍼灸院 - 公式ウェブサイト

Japanese acupuncture clinic website for "Wakando Shinkyu-in" (和漢堂 鍼灸院).

## Overview

A single-page, mobile-responsive website for a traditional Japanese acupuncture clinic. The design emphasizes calm, healing, and natural aesthetics using a teal/green color palette inspired by nature and traditional Japanese medicine.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main HTML document with all page sections |
| `style.css` | Stylesheet with responsive design and animations |

## Sections

1. **Header & Navigation** - Fixed top navigation with smooth scroll links
2. **Hero** - Calming gradient banner (#00695C to #80CBC4) with clinic tagline
3. **Treatment Menu** - Six core treatments: shoulder pain, lower back pain, headaches, cold sensitivity, cosmetic acupuncture, fertility support
4. **About Acupuncture** - Educational section on traditional Japanese acupuncture
5. **Therapist Profile** - Head therapist introduction with credentials
6. **Treatment Flow** - Step-by-step process: reception, consultation, treatment, aftercare
7. **Pricing Table** - Fee schedule for all treatments
8. **Testimonials** - Patient reviews
9. **Access & Hours** - Clinic location, map embed placeholder, and business hours
10. **Reservation Form** - Online booking form
11. **Footer** - Contact details and copyright

## Tech Stack

- Pure HTML5 and CSS3 (no JavaScript frameworks)
- Responsive design via CSS Grid, Flexbox, and media queries
- Google Fonts: Noto Serif JP, Noto Sans JP
- CSS custom properties for consistent theming

## Usage

Open `index.html` in any modern browser. No build step or server required.
