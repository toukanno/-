# 単位変換アプリ (Unit Converter)

A comprehensive unit converter web application with a clean Japanese UI.

## Features

- **8 conversion categories**: 長さ (Length), 重さ (Weight), 温度 (Temperature), 面積 (Area), 体積 (Volume), 速度 (Speed), 時間 (Time), データ容量 (Data)
- **Real-time conversion** as you type
- **Swap button** to reverse from/to units
- **Copy result** to clipboard
- **Conversion history** with recent entries
- **Mobile responsive** design
- **Japanese UI** throughout

## Usage

Open `index.html` in a browser. No build step or server required.

## File Structure

- `index.html` — Main HTML structure
- `style.css` — All styles and responsive layout
- `app.js` — Conversion logic, history, and UI interactions
