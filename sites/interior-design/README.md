# MUJI Design Studio

A minimalist Japanese interior design studio website built with pure HTML and CSS.

## Overview

MUJI Design Studio is a fictional interior design firm inspired by Japanese aesthetics — simplicity, natural materials, and intentional whitespace. The website reflects these principles through its clean layout and restrained color palette.

## Structure

- `index.html` — Single-page site with all sections
- `style.css` — Responsive styles with a minimalist Japanese aesthetic

## Sections

1. **Header** — Fixed navigation with smooth scroll links
2. **Hero** — Minimalist gradient background with studio tagline
3. **Works** — Portfolio grid showcasing residential and commercial projects
4. **Services** — Four service offerings (residential, office, retail, renovation)
5. **Process** — Step-by-step design workflow
6. **About** — Designer biography and philosophy
7. **Contact** — Inquiry form
8. **Footer** — Studio info and secondary navigation

## Design Choices

- Color palette: warm neutrals (`#f5f5f0`, `#d4c5a9`, `#2c2c2c`, `#8a7e6b`)
- Typography: system sans-serif stack for fast loading
- No JavaScript dependencies — CSS-only interactions and animations
- Fully responsive down to 320px viewports

## Running Locally

Open `index.html` in any modern browser. No build step or server required.
