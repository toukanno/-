# スマイル引越センター - Website

A single-page Japanese moving company website for **スマイル引越センター** (Smile Moving Center).

## Structure

- `index.html` - Main HTML page (single-page layout with all sections)
- `style.css` - All styles, fully responsive

## Sections

1. **Header** - Logo, navigation with smooth scroll links
2. **Hero** - Gradient banner (#FF6F00 to #FFA000) with call-to-action
3. **Services** - 4 service plans (単身パック, ファミリープラン, オフィス移転, 長距離引越し)
4. **Price Estimate** - Simple inline form for quick estimates
5. **Flow** - 5-step process (お見積り → 訪問見積り → 梱包 → 運搬 → 設置)
6. **Reviews** - Customer testimonials
7. **FAQ** - Accordion-style frequently asked questions
8. **Contact Form** - Free estimate request form
9. **Footer** - Company info, links, copyright

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.

## Design

- Bright, energetic orange gradient theme
- Trustworthy and professional tone
- Fully mobile-responsive (breakpoints at 768px and 480px)
- All content in Japanese
