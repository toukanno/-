# ピカピカクリーニング - 公式ウェブサイト

プロのハウスクリーニングサービス「ピカピカクリーニング」の公式ウェブサイトです。

## 概要

東京・神奈川・埼玉・千葉エリアを対象としたハウスクリーニングサービスのランディングページです。エアコン、浴室、キッチンなど各種クリーニングサービスの紹介、料金表、お客様の声、予約フォームを掲載しています。

## ファイル構成

| ファイル | 説明 |
|---|---|
| `index.html` | メインHTML（全セクション含む） |
| `style.css` | スタイルシート（レスポンシブ対応） |

## セクション一覧

1. **ヘッダー** - ロゴ・ナビゲーション（モバイルハンバーガーメニュー付き）
2. **ヒーロー** - キャッチコピーと予約誘導ボタン
3. **サービス紹介** - 6種類のクリーニングサービス
4. **ビフォーアフター** - 施工事例（プレースホルダー）
5. **料金表** - サービス別料金一覧
6. **サービスの流れ** - 予約から施工完了までの4ステップ
7. **お客様の声** - レビューカード
8. **対応エリア** - サービス提供地域のマップ
9. **予約フォーム** - お名前・連絡先・希望サービス・日時
10. **フッター** - 会社情報・リンク・著作権表示

## 技術仕様

- HTML5 / CSS3（外部ライブラリ不使用）
- レスポンシブデザイン（モバイル・タブレット・デスクトップ対応）
- グラデーション: `#00BCD4` → `#B2EBF2`
- 日本語フォント: sans-serif（デバイスデフォルト）

## ローカルでの確認方法

ブラウザで `index.html` を直接開くだけで動作します。サーバーは不要です。
