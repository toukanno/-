# アクアスイミングスクール 公式ウェブサイト

## 概要

アクアスイミングスクールの公式ウェブサイトです。ベビーから大人まで、幅広い年齢層に対応したスイミングレッスンを提供しています。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `index.html` | メインページ（全セクション含む） |
| `style.css` | スタイルシート |

## セクション一覧

1. **ヘッダー・ナビゲーション** - ロゴとページ内リンク、モバイルハンバーガーメニュー
2. **ヒーローセクション** - アクアグラデーション背景のメインビジュアル
3. **コース紹介** - ベビー、キッズ、ジュニア選手、大人、アクアビクスの5コース
4. **スケジュール** - 曜日別の時間割表
5. **施設紹介** - 25mプール、子供用プール、観覧席、更衣室・シャワー
6. **コーチ紹介** - 指導スタッフのプロフィール
7. **体験レッスン** - 無料体験の案内と申し込み誘導
8. **料金表** - コース別の月謝・入会金
9. **よくある質問（FAQ）** - アコーディオン形式のQ&A
10. **お問い合わせフォーム** - 名前・メール・電話・内容の入力フォーム
11. **フッター** - 所在地・営業時間・SNSリンク・著作権表示

## デザイン特徴

- **テーマカラー**: ウォーターブルー（`#0277BD` 〜 `#4FC3F7`）
- **レスポンシブ対応**: モバイル・タブレット・デスクトップ
- **外部依存なし**: フレームワーク不使用、Pure CSS + Vanilla JS
- **アクセシビリティ**: セマンティックHTML、適切なaria属性

## ローカルでの確認方法

```bash
# 任意のローカルサーバーで起動
cd sites/swim-school
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
