# サイクルショップ ペダル - Website

A responsive Japanese bicycle shop website for "サイクルショップ ペダル" (Cycle Shop Pedal).

## Overview

This is a single-page website for a fictional bicycle shop located in Tokyo. The site features a sporty, clean design with a blue accent color scheme.

## Pages and Sections

- **Header**: Navigation bar with shop logo and menu links
- **Hero**: Full-width gradient banner with call-to-action
- **Products**: Showcase of five bicycle categories (road, cross, mountain, e-assist, kids)
- **Repair Services**: Four core repair/maintenance services offered
- **Staff**: Introduction of shop staff members
- **Access & Hours**: Shop location, map placeholder, and business hours
- **Contact**: Inquiry form with name, email, category, and message fields
- **Footer**: Copyright and secondary navigation

## Tech Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript (mobile menu toggle only)
- Fully responsive design (mobile-first)
