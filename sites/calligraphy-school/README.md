# 翠風書道教室 - Suifu Calligraphy School Website

A minimalist, elegant website for a Japanese calligraphy school (書道教室).

## Overview

This is a single-page website for 翠風書道教室 (Suifu Shodo Kyoshitsu), a Japanese calligraphy school. The design emphasizes a clean, traditional aesthetic with modern responsiveness.

## Files

- `index.html` — Main HTML document containing all page sections
- `style.css` — Stylesheet with responsive design, ink-inspired gradients, and Japanese aesthetic

## Sections

1. **Header / Navigation** — Fixed top navigation with smooth scroll links
2. **Hero** — Full-screen ink gradient background with school name and tagline
3. **About (書道の魅力)** — Introduction to the art of calligraphy
4. **Courses (コース案内)** — Four course offerings: children, general, pen, and trial lessons
5. **Works Gallery (作品紹介)** — Showcase of student and teacher works
6. **Teacher Profile (講師紹介)** — Instructor biography and credentials
7. **Schedule & Pricing (スケジュール・料金)** — Class times and fee structure
8. **Access (アクセス)** — Location and directions
9. **Contact (お問い合わせ)** — Inquiry form
10. **Footer** — Copyright and links

## Design

- **Color palette**: Black (`#212121`), dark gray (`#616161`), white, gold accents (`#b8860b` / `#d4a537`)
- **Typography**: Noto Serif JP for headings, Noto Sans JP for body text
- **Layout**: Mobile-first responsive design
- **Style**: Minimalist with Japanese brush art influences

## Usage

Open `index.html` in any modern web browser. No build tools or server required.
