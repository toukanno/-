# さくら動物病院 - Sakura Animal Hospital Website

A modern, responsive website for a Japanese veterinary clinic.

## Structure

- `index.html` - Main HTML page with all sections
- `style.css` - Stylesheet with CSS variables, responsive design, and modern layout

## Features

- Fully responsive layout for mobile, tablet, and desktop
- CSS custom properties for consistent theming
- Warm gradient hero section
- Services grid with icons
- Doctor profile section
- Business hours schedule table
- Emergency contact information
- Access/map section with embedded Google Maps placeholder
- Contact form
- Smooth scroll navigation

## Sections

1. Header with navigation (ホーム, 診療案内, 院長紹介, 施設紹介, アクセス)
2. Hero section with clinic tagline
3. Services (一般診療, 予防接種, 健康診断, 手術, トリミング, ペットホテル)
4. Doctor profile (院長紹介)
5. Hours/schedule table (診療時間)
6. Emergency info (緊急時のご案内)
7. Facility introduction (施設紹介)
8. Access map (アクセス)
9. Contact form (お問い合わせ)
10. Footer

## Usage

Open `index.html` in any modern browser. No build step or dependencies required.
