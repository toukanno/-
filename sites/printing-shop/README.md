# プリントワークス - PrintWorks Website

A professional Japanese printing company website for プリントワークス.

## Overview

This is a single-page responsive website for a Japanese printing shop offering various print services including business cards, flyers, posters, pamphlets, envelopes, and stickers.

## Files

- `index.html` - Main HTML file with all page sections
- `style.css` - Complete stylesheet with responsive design

## Sections

1. **Header** - Navigation bar with logo and menu links
2. **Hero** - Creative gradient banner (#E91E63 to #F48FB1) with call-to-action
3. **Services** - Six printing services with icons and descriptions
4. **Price List** - Table of pricing for each service tier
5. **Order Flow** - Visual 4-step process (データ入稿 → 確認 → 印刷 → 発送)
6. **Quality Commitment** - Company values and quality assurance
7. **FAQ** - Frequently asked questions with accordion-style answers
8. **Order/Inquiry Form** - Contact and order request form
9. **Footer** - Company info, links, and copyright

## Features

- Fully responsive (mobile, tablet, desktop)
- Japanese language throughout
- Creative, colorful, and professional design
- CSS-only interactions (no JavaScript frameworks required)
- Accessible semantic HTML

## Usage

Open `index.html` in any modern web browser. No build step or server required.
