# プリントワークス - PrintWorks Website

A professional website for プリントワークス, a Japanese printing company.

## Overview

This is a single-page responsive website built with HTML and CSS (no JavaScript frameworks). It showcases the company's printing services, pricing, order flow, and includes a contact/inquiry form.

## Files

- `index.html` - Main HTML page
- `style.css` - All styles including responsive design

## Sections

1. **Header** - Navigation bar with company logo and menu links
2. **Hero** - Eye-catching gradient banner (#E91E63 to #F48FB1)
3. **Services** - Six printing services offered (名刺印刷, チラシ・フライヤー, ポスター, パンフレット, 封筒・はがき, シール・ステッカー)
4. **Price List** - Pricing table for each service
5. **Order Flow** - Four-step process (データ入稿 → 確認 → 印刷 → 発送)
6. **Quality Commitment** - Company quality promises
7. **FAQ** - Frequently asked questions
8. **Order/Inquiry Form** - Contact and order form
9. **Footer** - Company info, links, and copyright

## Features

- Fully responsive (mobile, tablet, desktop)
- Creative gradient color scheme
- Professional Japanese typography
- CSS-only interactions (no JavaScript required for layout)
- Accessible semantic HTML

## Usage

Open `index.html` in any modern web browser.
