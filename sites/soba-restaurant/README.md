# 蕎麦処 風月庵 - Website

A static website for a traditional Japanese soba restaurant, "Fugetsuan" (風月庵).

## Files

- `index.html` - Main HTML page (fully in Japanese)
- `style.css` - Stylesheet with traditional Japanese aesthetic, responsive design

## Features

- Responsive layout for mobile, tablet, and desktop
- Traditional warm earth-tone colour palette
- Sections: navigation, hero, menu (品書き), craft/kodawari (こだわり), about, access, footer
- No external dependencies; pure HTML and CSS

## Viewing

Open `index.html` in any modern browser.
