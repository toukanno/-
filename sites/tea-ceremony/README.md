# 裏千家 松風庵 - Japanese Tea Ceremony School Website

A website for Urasenke Shofuan, a traditional Japanese tea ceremony school.

## Overview

This is a static website for a Japanese tea ceremony school (茶道教室) called 松風庵 (Shofuan), affiliated with the Urasenke (裏千家) tradition. The site provides information about the school, lesson courses, schedule, pricing, teacher profile, access, and a contact form.

## Files

- `index.html` - Main HTML page with all sections
- `style.css` - Stylesheet with traditional Japanese aesthetic, bamboo/matcha color scheme, and mobile responsive design

## Sections

1. **Header** - Navigation bar with school name and links
2. **Hero** - Traditional gradient banner (#2d5016 to #6b8e23) with school tagline
3. **About** - Introduction to the way of tea (茶道について)
4. **Courses** - Four lesson tiers: 初心者コース, 中級コース, 上級コース, 体験レッスン
5. **Schedule & Pricing** - Weekly schedule and fee structure
6. **Gallery** - Tea room atmosphere (茶室の様子) with placeholder images
7. **Teacher Profile** - Instructor biography
8. **Access** - Location and directions
9. **Contact** - Inquiry form
10. **Footer** - Copyright and supplementary links

## Design

- Traditional Japanese aesthetic with elegant serif-like typography
- Bamboo and matcha green color scheme
- Fully mobile responsive
- CSS-only (no JavaScript framework dependencies)

## Usage

Open `index.html` in any modern web browser. No build step or server required.
