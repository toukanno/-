# カラーピッカー & パレットツール

モダンなカラーピッカー・パレット生成ツールです。

## 機能

- **カラーピッカー**: 色相・彩度・明度キャンバスによる直感的な色選択
- **カラー入力**: RGB、HSL、HEX の相互同期入力フィールド
- **カラープレビュー**: 選択中の色を大きなスウォッチで表示
- **コピー機能**: HEX、RGB、HSL 値をワンクリックでコピー
- **カラーハーモニー**: 補色・類似色・トライアドの表示
- **パレット生成**: 5色のカラーパレットを自動生成
- **保存機能**: localStorage を使ったカラー保存
- **グラデーション生成**: 2色ストップ、方向指定、CSS出力
- **コントラストチェッカー**: WCAG AA/AAA 基準の判定
- **プリセットパレット**: マテリアルデザイン、パステルなど

## 使い方

`index.html` をブラウザで開くだけで使えます。サーバー不要です。

## 技術

- HTML5 Canvas
- Vanilla JavaScript (フレームワーク不使用)
- CSS Grid / Flexbox
- localStorage API
- Clipboard API
