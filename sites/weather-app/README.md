# Weather Dashboard / 天気ダッシュボード

モックデータを使用した美しいガラスモーフィズムデザインの天気ダッシュボードWebアプリです。

## 機能

- 都市検索バー
- 現在の天気表示（気温、湿度、風速、天気アイコン）
- 7日間の天気予報カード
- 時間ごとの予報グラフ（CSSバー）
- Unicode/絵文字による天気アイコン
- 温度単位切替（℃/℉）
- プリセット都市（東京、大阪、名古屋、札幌、福岡、沖縄）
- 天気に応じた背景グラデーション変化
- 現在の日時表示
- ガラスモーフィズムデザイン
- モバイルレスポンシブ対応
- 日本語UI

## 使い方

`index.html` をブラウザで開くだけで動作します。APIキーは不要です（モックデータ使用）。

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイルシート
- `app.js` - アプリケーションロジック・モックデータ
