# 麺屋 龍 (Menya Ryu) - Ramen Shop Website

A responsive, single-page Japanese ramen shop website built with semantic HTML5 and modern CSS.

## Features

- Fully Japanese-language content
- Mobile-responsive layout with sticky navigation
- CSS custom properties for consistent theming
- Menu cards, features section, access/map info, and contact form
- Professional design inspired by Japanese restaurant sites

## Files

- `index.html` - Main HTML document
- `style.css` - All styles including responsive breakpoints

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.
