# Markdown Editor

A fully-featured Markdown editor web application with live preview, built with vanilla HTML, CSS, and JavaScript (no external libraries).

## Features

- **Split-pane layout**: Editor on the left, live preview on the right
- **Custom Markdown parser**: Headers, bold, italic, strikethrough, lists, code blocks, inline code, links, images, blockquotes, horizontal rules
- **Toolbar**: Quick formatting buttons
- **Word/character count**: Real-time statistics
- **Export as HTML**: Download rendered HTML
- **Copy Markdown**: Copy raw markdown to clipboard
- **Dark/light theme toggle**: Persistent theme preference
- **LocalStorage auto-save**: Never lose your work
- **Full-screen mode**: Distraction-free editing
- **Mobile responsive**: Tab-based UI on small screens
- **Japanese UI**: Interface in Japanese

## Usage

Open `index.html` in any modern browser. No build step or server required.

## Markdown Syntax Supported

| Syntax          | Example                    |
|-----------------|----------------------------|
| Headers         | `# H1`, `## H2`, `### H3` |
| Bold            | `**bold**`                 |
| Italic          | `*italic*`                 |
| Strikethrough   | `~~strikethrough~~`        |
| Unordered list  | `- item`                   |
| Ordered list    | `1. item`                  |
| Code block      | `` ``` code ``` ``         |
| Inline code     | `` `code` ``               |
| Link            | `[text](url)`              |
| Image           | `![alt](url)`              |
| Blockquote      | `> quote`                  |
| Horizontal rule | `---`                      |
