# ワインバー ボルドー - Wine Bar Bordeaux

A luxurious Japanese wine bar website featuring an elegant dark theme with purple and gold accents.

## Overview

This is a single-page website for "ワインバー ボルドー" (Wine Bar Bordeaux), a fictional upscale wine bar in Tokyo. The site is fully responsive and built with vanilla HTML and CSS.

## Sections

- **Header** - Navigation bar with smooth scroll links
- **Hero** - Full-screen gradient hero with tagline
- **Wine List** - Cards for red, white, sparkling, and rose wines
- **Food Menu** - Pairing menu items with descriptions
- **Sommelier** - Staff profile section
- **Interior** - Atmosphere and ambiance showcase
- **Events** - Wine tasting events and seasonal gatherings
- **Reservation** - Booking form with date, time, and party size
- **Access & Hours** - Location, map placeholder, and business hours
- **Footer** - Copyright, social links, and contact info

## Tech

- HTML5 / CSS3
- No JavaScript frameworks required (minimal vanilla JS for mobile menu toggle)
- Google Fonts: Noto Serif JP, Cormorant Garamond
- Responsive down to 320px

## Usage

Open `index.html` in any modern browser. No build step required.
