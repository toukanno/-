# QRコード生成ツール

ブラウザ上で動作するQRコード生成Webアプリケーションです。外部CDNやライブラリに依存せず、完全にセルフコンテインドで動作します。

## 機能

- **複数入力タイプ対応**: URL、テキスト、メール、電話番号、WiFi
- **QRコードカスタマイズ**: サイズ（小・中・大）、前景色・背景色の変更
- **エクスポート**: PNG画像としてダウンロード、印刷機能
- **レスポンシブデザイン**: モバイル・デスクトップ両対応
- **日本語UI**: 全てのインターフェースが日本語

## ファイル構成

| ファイル | 説明 |
|---|---|
| `index.html` | メインHTML |
| `style.css` | スタイルシート |
| `app.js` | QRコード生成ロジック・UIロジック |

## 使い方

1. `index.html` をブラウザで開く
2. 入力タイプを選択（URL、テキスト等）
3. 内容を入力
4. 「QRコード生成」ボタンをクリック
5. 必要に応じてサイズ・色を調整
6. ダウンロードまたは印刷

## 技術詳細

QRコード生成は純粋なJavaScriptで実装されています（外部依存なし）。QRコードのエンコーディングにはISO/IEC 18004規格に基づくアルゴリズムを使用し、Canvas APIでレンダリングします。
