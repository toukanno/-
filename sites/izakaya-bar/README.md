# 酒場 月見 (Sakaba Tsukimi) - Izakaya Website

A single-page website for a Japanese izakaya bar called 酒場 月見 (Sakaba Tsukimi).

## Structure

- `index.html` — Main HTML page with all sections (hero, menu, drinks, courses, atmosphere, access, reservation)
- `style.css` — Stylesheet with dark theme, warm colour palette, and responsive layout

## Features

- Dark, atmospheric design inspired by traditional izakaya interiors
- Full food and drink menu with pricing
- All-you-can-drink course menu section (飲み放題コース)
- Reservation form
- Access information with business hours
- Fully responsive for mobile, tablet, and desktop

## Running

Open `index.html` in any modern browser. No build tools or server required.
