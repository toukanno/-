# Todo App

モダンなデザインのTodoアプリケーション。

## 機能

- タスクの追加・編集・削除
- チェックボックスで完了マーク（取り消し線表示）
- 優先度レベル（高・中・低）のカラーコーディング
- 期日ピッカー
- カテゴリ・タグ機能
- フィルター（すべて・未完了・完了済み）
- 検索バー
- ドラッグ＆ドロップによる並べ替え
- LocalStorageによるデータ永続化
- 完了済みタスクの一括削除
- タスク数の表示
- ダークモード切替
- モバイルレスポンシブ対応

## 使い方

`index.html` をブラウザで開くだけで使用できます。サーバー不要です。

## 技術

- HTML5 / CSS3 / Vanilla JavaScript
- LocalStorage API
- HTML5 Drag and Drop API
