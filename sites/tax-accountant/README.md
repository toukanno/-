# 山田税理士事務所 公式ウェブサイト

山田税理士事務所の公式ウェブサイトのソースコードです。

## 概要

東京都千代田区を拠点とする税理士事務所のウェブサイトです。法人・個人のお客様に対し、税務申告、経営相談、会社設立支援などのサービスを提供しています。

## ファイル構成

```
tax-accountant/
├── README.md      # このファイル
├── index.html     # メインHTMLファイル
└── style.css      # スタイルシート
```

## セクション構成

- **ヘッダー・ナビゲーション**: ロゴとページ内リンク
- **ヒーロー**: メインビジュアル（グリーン系グラデーション）
- **サービス紹介**: 法人税、所得税、相続税、確定申告、経営相談、会社設立
- **事務所紹介**: 代表プロフィールと理念
- **ご相談の流れ**: お問い合わせから契約までの4ステップ
- **よくあるご質問**: FAQ形式
- **料金案内**: サービス別の料金表
- **お問い合わせ**: コンタクトフォーム
- **フッター**: 事務所情報とリンク

## 技術仕様

- HTML5 / CSS3（外部ライブラリ不使用）
- レスポンシブデザイン対応（モバイル・タブレット・デスクトップ）
- 日本語フォント: Google Fonts（Noto Sans JP）

## 表示方法

`index.html` をブラウザで開くだけで表示できます。サーバー環境は不要です。
