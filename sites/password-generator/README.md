# Password Generator

A sleek, dark-themed password generator web application with a hacker aesthetic and Japanese UI.

## Features

- **Password length slider** (4-64 characters)
- **Character options**: uppercase, lowercase, numbers, symbols (toggleable)
- **Exclude ambiguous characters** option (e.g., `0O`, `1lI`, `|`)
- **Password strength meter** with bar and Japanese labels
- **Generate button** for single password
- **Generate multiple passwords** (5 at a time)
- **Copy to clipboard** functionality
- **Password history** (last 10, persisted in localStorage)
- **Estimated crack time** display
- **Mobile responsive** design

## Usage

Open `index.html` in any modern browser. No build step or server required.

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript (no dependencies)
